Fridge ERD

ERD Description

Table FoodType

  • This table represents the different types of food there are
  • The table consists of a Primary key (Id) for each FoodTypeName and of the FoodTypeName attribute, which represents a certain type of food there is.
  • The Primary key in this table is used in the Food table as a Foreign key to indicate what type of food a certain food is.

Table Food

  • This table represents what food there currently is
  • The table consists of a Primary key (Id) for each FoodName, of FoodName which simply says the name of the food. It also has a Foreign key called TypeId that shows what type of food a certain food is.
  • The Foreign key in this table is from the FoodType table and it indicates what type of food a certain food is.
